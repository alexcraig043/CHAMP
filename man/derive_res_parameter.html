derive_res_parameterR Documentation

Derive Resolution Parameter

Description

This function computes the resolution parameter based on a given partition and network. The resolution parameter is computed only if there is more than one cluster in the partition.

Usage

derive_res_parameter(partition, network)

Arguments

partition

A list containing details of the partition. It should include 'nb_clusters' for the number of clusters, and 'membership' for the membership of each element.

network

The network graph on which the partition is placed.

Value

A numeric value representing the resolution parameter, or NULL if there's only one cluster.

Examples

# Assume you have a partition object and a network object defined
derive_res_parameter(partition, network)
