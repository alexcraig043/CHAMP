CHAMP function

Description

This function identifies the Convex Hull of Admissible Modularity Partitions (CHAMP) set of partitions and their corresponding domains of optimality.

Usage

CHAMP(network, partitions, plottitle = NULL)

Arguments

network

The network to analyze.

partitions

The list of partitions to consider.

plottitle

The title of the plot to generate (optional).

Value

A summary of the CHAMP partitions, including the range of gamma values for which each partition is part of the CHAMP set and the number of clusters in each partition.

Author(s)

Code written in parts by Rachel Matthew, Ryan Rebne, Ava Scharfstein and PJM.
