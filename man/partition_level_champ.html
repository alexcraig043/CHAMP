partition_level_champR Documentation

Calculate partition level metrics for CHAMP

Description

Computes the metrics 'a_partition' and 'p_partition' based on the given network and partition.

Usage

partition_level_champ(network, partition)

Arguments

network

The network to be analyzed.

partition

A list representing the partitioning of the network's vertices.

Value

A list containing the values for 'a_partition' and 'p_partition'.

Examples

result <- partition_level_champ(network, partition)
