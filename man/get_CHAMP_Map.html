get_CHAMP_MapR Documentation

Wrapper for Other CHAMP_Map_scripts Helper Functions to Get the Iterative Map

Description

This function is a wrapper for other CHAMP_Map_scripts helper functions to get the iterative map for the resolution parameter on the CHAMP set of partitions and their corresponding domains of optimality.

Usage

get_CHAMP_Map(network, partitions, partition_summary, plottitle = NULL)

Arguments

network

A network object representing the structure of the network.

partitions

A list containing the different partitions.

partition_summary

A data frame containing the summary of the partitions, obtained by CHAMP.R.

plottitle

(Optional) A title for the plot.

Value

A data frame containing the updated partition_summary with iterative map information for all partitions.
