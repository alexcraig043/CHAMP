find_maxR Documentation

Find Maxima in Modularity Matrix

Description

This function identifies points of change and corresponding partitions based on a given modularity matrix and range of gamma values.

Usage

find_max(mod_matrix, gammas)

Arguments

mod_matrix

A matrix containing 'base' and 'decrement' columns representing the modularity structure.

gammas

A numeric vector representing the range of gamma values to consider.

Value

A list with two elements: 'edges', containing the points of change, and 'partitions', containing the corresponding partitions.

Examples

# Assume mod_matrix is defined with appropriate 'base' and 'decrement' columns
# and gammas is defined as a range of values
find_max(mod_matrix, gammas)
